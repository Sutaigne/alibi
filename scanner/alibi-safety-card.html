Alibi · Safety Card

Is this scan safe to run?

A one-page attestation of what the scan does and doesn't do. Written for the reviewer who's being sent a report and wants to know the kit itself isn't a risk.

Short answer
Yes. The scan is read-only, never touches the internet, and only ever creates one file: the report itself.
In one picture

What goes in, what comes out, what doesn't happen at all

The scan reads from your PC, scores what it finds, and writes one text report to your Desktop. Nothing leaves the machine. The script doesn't open a network connection, doesn't download or install anything, and doesn't change a single system setting. It runs once, writes the report, and exits.

The specifics

Exactly what it does and exactly what it doesn't

What it does

Read, score, write one file

  • Reads your running process list (same data Task Manager shows)
  • Reads your installed-software list (same data the Settings app shows)
  • Reads USB device history (which devices have been plugged in over time)
  • Reads file names and paths only — never opens the inside of any file
  • Compares what it finds against a list of known cheat / input-device names
  • Writes one .txt file to your Desktop with a timestamped name
  • Exits
What it doesn't do

None of the things you'd reasonably worry about

  • Makes one opt-in outbound call — the LOLDrivers (loldrivers.io) BYOVD cross-reference — which prompts Y/N before running and is the only Invoke-WebRequest in the kit. No telemetry, no phone-home, no cloud upload. Nothing about your PC is sent — only the public driver database is downloaded.
  • Doesn't download or install anything
  • Doesn't modify any existing file, registry key, service, or setting
  • Doesn't read the contents of your photos, documents, messages, or saved passwords
  • Doesn't run in the background after it finishes — no scheduled task, no startup entry, no service
  • Doesn't send the report anywhere automatically. It just sits on your Desktop until you choose to send it
  • Doesn't require any external software, downloads, or accounts
Don't take my word for it

Three ways to verify any of this yourself

VERIFY 01

Open the script in Notepad

Right-click forensic-scan.ps1 → Open with → Notepad. Every operation the script performs is visible as plain English plus PowerShell. Search scanner\*.ps1 for Invoke-Web and you'll find exactly one result — the opt-in LOLDrivers fetch on forensic-common.ps1 line 736, prompted with Y/N at runtime. Search for Remove- or Set-Item and you'll find none in the scan path.

VERIFY 02

Watch network activity while it runs

Open Task Manager → Performance → Ethernet / Wi-Fi. Start the scan. If you press anything other than Y at the LOLDrivers prompt, the "Send" graph stays at zero kbps for the entire run. If you press Y, you'll see a single small fetch (~50 KB) from loldrivers.io and then nothing else. There are no other outbound connections.

VERIFY 03

Look at the output yourself

The .txt report on the Desktop is the only thing the scan creates. Open it in Notepad. Every line is human-readable. There is no hidden second file anywhere on the system.

The scan is roughly 950 lines of PowerShell. The whole script is plain text in a single file you can read top to bottom. There is no compiled binary, no DLL, no installer, no obfuscation. If something the script does isn't visible in the script, the script doesn't do it.
